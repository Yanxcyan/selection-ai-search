# 划词AI搜索

选中网页上的任意文字，按一下快捷键，页面上就会弹出一个小窗口，用你自己配置的 AI（DeepSeek / Claude / OpenAI 官方，或任意 OpenAI 兼容中转站）帮你解释/回答，全程不用跳转、不用复制粘贴到别的网站。

## 功能点

- **划词 + 快捷键触发**：鼠标拖选文字后按快捷键（默认 `Alt+Q`），直接弹小窗，不跳转页面、不打断阅读
- **快捷键可自定义**：设置面板里点一下输入框，直接按你想要的组合键，自动捕获保存
- **多 AI 供应商，自己配置**：
  - 内置 DeepSeek 官方 / Claude 官方 / OpenAI 官方 三个预设
  - 外加一个「自定义中转站」，填你自己的中转地址 + Key + 模型名就能用
  - 弹窗顶部可以随时切换当前用哪个 AI，切换后自动重新提问
- **三种回复风格**：丰富 / 简短 / 仅答案，弹窗里和设置里都能切
- **流式输出**：像正常聊天一样一个字一个字往外蹦，不用死等一整段返回
- **弹窗可拖拽**，支持重试、一键复制结果、`Esc` 关闭
- **Shadow DOM 隔离样式**：不会被各种网站的全局 CSS 污染，暗色 UI 到哪个网站都长一个样

## 安装方法

先装一个 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展，然后任选一种方式安装脚本：

**方式一（推荐）**：点开 [`selection-ai-search.user.js`](./selection-ai-search.user.js)，再点页面右上角的 **Raw** 按钮，Tampermonkey 会自动识别并弹出安装确认页，点安装即可。

**方式二（手动粘贴）**：
1. 点开 Tampermonkey 扩展菜单，选「新建脚本」
2. 打开 [`selection-ai-search.user.js`](./selection-ai-search.user.js)，把里面的内容整个复制，替换掉新建脚本里的默认内容
3. 保存

安装完成后：
1. 打开任意网页，Tampermonkey 菜单里点「划词AI搜索 - 设置」，填上你的 API Key
2. 选中一段文字，按 `Alt+Q` 试试

## 配置说明

设置面板里可以填：

| 字段 | 说明 |
|---|---|
| 触发快捷键 | 默认 `alt+q`，点输入框后按键自动捕获 |
| 回复风格 | 丰富 / 简短 / 仅答案，三选一 |
| 默认使用的 AI | 从下面配置的供应商列表里选一个 |
| 接口地址 | 完整的 chat completions 接口地址，**注意要带路径**，比如 `https://xxx.com/v1/chat/completions`，不是只填域名 |
| API Key | 对应供应商的密钥 |
| 模型名 | 比如 `deepseek-chat`、`gpt-4o-mini`、`claude-sonnet-5`，中转站要填它文档里支持的模型名 |
| 协议类型 | OpenAI 兼容（DeepSeek/OpenAI/大多数中转站）或 Anthropic（Claude 官方接口） |

用中转站的话记得选对协议类型、填对模型名，不然容易出现"接口通了但报错/无响应"的情况。

## 已知限制

- 无法在跨域 iframe 里生效（比如某些网页的正文其实是嵌在 iframe 里的，油猴脚本进不去）
- 中转站是否好用取决于它本身是否稳定支持流式返回和标准 OpenAI/Anthropic 协议
- 目前没做多轮对话/上下文记忆，每次划词都是独立的一次提问
