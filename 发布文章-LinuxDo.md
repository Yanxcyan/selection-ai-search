# 【油猴脚本】划词AI搜索 —— 选中文字按快捷键，不离开当前页面直接问 AI，自由配置官方/中转 API

## 一句话介绍

用鼠标选中网页上的任意文字，按一下快捷键（默认 `Alt+Q`，可自定义），页面右侧就会弹出一个小窗口，用你自己配置的 AI（DeepSeek / Claude / OpenAI 官方，或任意 OpenAI 兼容中转站）帮你解释/回答，全程不用跳转、不用复制粘贴到别的网站。

写这个东西纯粹是因为自己看文档、看论坛的时候经常需要划词丢给 AI 问一下，来回切标签页太烦，干脆自己写一个。顺手发出来，看看有没有人也用得上。

---

## 功能点

- **划词 + 快捷键触发**：鼠标拖选文字后按快捷键，直接弹小窗，不跳转页面、不打断阅读
- **快捷键可自定义**：设置面板里点一下输入框，直接按你想要的组合键，自动捕获保存
- **多 AI 供应商，自己配置**：
  - 内置 DeepSeek 官方 / Claude 官方 / OpenAI 官方 三个预设
  - 外加一个「自定义中转站」，填你自己的中转地址 + Key + 模型名就能用
  - 弹窗顶部可以随时切换当前用哪个 AI，切换后自动重新提问
- **三种回复风格**：丰富 / 简短 / 仅答案，看心情选，弹窗里和设置里都能切
- **流式输出**：像正常聊天一样一个字一个字往外蹦，不用死等一整段返回
- **弹窗可拖拽**，支持重试、一键复制结果、`Esc` 关闭
- **Shadow DOM 隔离样式**：不会被各种奇奇怪怪的网站全局 CSS 污染，暗色 UI 到哪个网站都长一个样

## 安装方法

1. 装一个 Tampermonkey（油猴）浏览器扩展
2. 新建一个脚本，把下面 `.user.js` 的内容整个粘贴进去，保存
3. 打开任意网页，Tampermonkey 菜单里点「划词AI搜索 - 设置」，填上你的 API Key
4. 选中一段文字，按 `Alt+Q` 试试

> 也可以把 `.user.js` 文件直接拖进已安装 Tampermonkey 的浏览器窗口，会自动弹出安装确认。

## 配置说明

设置面板里可以填：

| 字段 | 说明 |
|---|---|
| 触发快捷键 | 默认 `alt+q`，点输入框后按键自动捕获 |
| 回复风格 | 丰富 / 简短 / 仅答案，三选一 |
| 默认使用的 AI | 从下面配置的供应商列表里选一个 |
| 接口地址 | 完整的 chat completions 接口地址，**注意要带路径**，比如 `https://xxx.com/v1/chat/completions`，不是只填域名 |
| API Key | 对应供应商的密钥 |
| 模型名 | 比如 `deepseek-chat`、`gpt-4o-mini`、`claude-sonnet-5`，中转站要填它文档里支持的模型名 |
| 协议类型 | OpenAI 兼容（DeepSeek/OpenAI/大多数中转站）或 Anthropic（Claude 官方接口） |

用中转站的话记得选对协议类型、填对模型名，不然容易出现"接口通了但报错/无响应"的情况。

## 已知限制

- 无法在跨域 iframe 里生效（比如某些网页的正文其实是嵌在 iframe 里的，油猴脚本进不去）
- 中转站是否好用取决于它本身是否稳定支持流式返回和标准 OpenAI/Anthropic 协议
- 目前没做多轮对话/上下文记忆，每次划词都是独立的一次提问
